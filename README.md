**MaybeTomorrow** - проект, нацеленный на облегчение планирования событий.

> Проект направлен на создание интерактивного календаря, на котором можно будет отмечать занятое/свободное время (день) и делиться этой информацией. Пользователи смогут создавать события, привязывать их к определенному дню (дням), устанавливать дедлайн событий. Также пользователи смогут делиться своими календарями, с целью «совмещения» двух календарей для выявления свободного/занятого времени у обоих пользователей.

#### Цель проекта

Разработать веб-приложение (сайт и серверную часть), которое позволяет отмечать занятое и свободное время на интерактивном календаре и делиться данной информацией с группой других пользователей с целью выявления свободного времени у группы пользователей

#### Актуальность проекта

Лично для меня, как для автора проекта, актуальность заключается в необходимости выбора удобного дня, когда все мои товарищи будут свободны, чтобы мы вместе могли поиграть в какие-нибудь игры.
Если же взять уровень университета, в котором я обучаюсь, то иногда у нескольких людей возникает необходимость сравнивать свои расписания, чтобы выбрать удобный для всех день, в который можно будет провести встречу.
Примером является следующее: двум куратором необходимо сверить свое расписание и расписание их групп, чтобы провести совместный тренинг и тому подобные мероприятия

#### Целевая аудитория

Геймеры, кураторы групп (команд), члены различных объединений (групп), студенты и обучающиеся

### Предполагаемый стек технологий

Frontend: Typescript + SvelteKit (Sveltejs), Tailwindcss, Bootstrap Icons
Backend: Golang (fiber)
Database: PostgreSQL

#### Предполагаемые риски

- Рыночные риски: отсутствие заинтересованности целевой аудитории в продукте, конкуренция
- Временные риски: отсутствие свободного времени у команды разработки (из-за высокой учебной нагрузки)
- Квалификационные риски: недостаточность опыта у команды разработки
- Технические риски: сложность реализации логики проекта

### MVP

Минимальная жизнеспособная версия продукта включает в себя:

1. Регистрацию и создание профилей пользователей
2. Систему добавления мероприятий и событий с указанием описания, даты и времени начала и окончания выполнения, периодичностью (повторением) определенного события в различное время (дни)
3. Календарный вид с отображением запланированных мероприятий и событий
4. Возможность редактирования и удаления записей о событиях
5. Функцию «совмещения» двух и более расписаний
6. Функцию суммаризации расписаний нескольких пользователей (в режиме «совмещения» будет показывать свободные для всех пользователей дни)
7. Форму обратной связи (пятибалльную шкалу оценки и пожелания)
